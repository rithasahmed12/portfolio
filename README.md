# Rithas Ahmed - Developer Portfolio

A modern, animated personal portfolio for **Rithas Ahmed**, a Full Stack Software Engineer.
Built with Next.js and Tailwind CSS, featuring a 3D particle background, smooth Framer Motion
animations, polished light/dark themes, an interactive project gallery, a working contact form,
and a downloadable resume.

**Live:** https://rithas.vercel.app

---

## Features

- **3D animated background** - floating particles rendered with Three.js / React Three Fiber.
- **Light & dark themes** - a cohesive "Soft Slate + Teal" light theme that mirrors the deep
  navy + cyan dark theme, with a one-click toggle.
- **Motion-rich UI** - section reveals, magnetic buttons, and a custom cursor glow via Framer Motion.
- **Sections** - Hero, About, Skills, Projects, Experience, and Contact.
- **Interactive project gallery** - each project opens a modal with an image carousel, highlights,
  tech stack, and GitHub / live links.
- **Working contact form** - enquiries are delivered straight to my inbox via Web3Forms
  (with loading / success / error states and spam protection).
- **Floating WhatsApp button** - always-visible quick-contact shortcut.
- **Resume preview & download** - view the resume in an inline modal or download the PDF.
- **Fully responsive** - tuned down to 320px-wide screens.
